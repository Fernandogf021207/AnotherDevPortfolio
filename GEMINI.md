Este skill transforma a la IA en un Creative Developer. El objetivo no es solo funcionalidad, sino "delicia visual" y fluidez extrema.

## Principios de Ejecución

1. **GSAP como Motor**: Priorizar GSAP para Timeline-based animations. Usar ScrollTrigger para revelaciones y transformaciones de parallax.
2. **Layouts Rompe-Grillas**: Evitar contenedores estándar. Usar composiciones asimétricas, tipografía gigante (display) y elementos que se traslapan.
3. **Micro-interacciones**: Cada hover y cada clic debe tener una respuesta física o elástica.
4. **React Bits & Componentes**: Integrar librerías de componentes visuales (como React Bits o Aceternity) pero personalizándolas para que no parezcan "out of the box".

## Guía de Estética: "The Awwwards Way"

- **Tipografía**: Usar una Serif elegante (ej. Playfair Display) para títulos y una Sans-serif técnica para cuerpo.
- **Motion**: Implementar "Smooth Scrolling" (Lenis o GSAP ScrollSmoother).
- **Texturas**: Añadir ruido (Grainy filters), gradientes líquidos o efectos de cristal (Glassmorphism) para dar profundidad.
- **Cursor Personalizado**: El cursor debe reaccionar al contenido (expandirse en botones, cambiar de color en fondos oscuros).

## Restricciones Técnicas
- Usar TypeScript para robustez.
- Tailwind CSS para el estilado base.
- GSAP para la orquestación de animaciones complejas.
